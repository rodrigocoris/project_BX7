# SP Accesorios ERP

Proyecto base de ERP moderno para SP Accesorios, creado con React, TypeScript, Vite, Framer Motion y Recharts.

## Incluye

- Login de demo con sesión persistente en `localStorage`.
- Panel administrativo con resumen ejecutivo.
- Gestión visual de proveedores y productos.
- Bloque de predicción de demanda y actividad reciente.
- Animaciones y una interfaz responsive con estética moderna.

## Credenciales de demo

- Email: `admin@spaccesorios.com`

## Arranque

1. Instala Node.js 20 o superior.
2. Ejecuta `npm install` en la carpeta del proyecto.
3. Ejecuta `npm run dev`.

## Nota técnica

La base está preparada como frontend de administrador. Si quieres, el siguiente paso natural es conectar una API real, base de datos, roles y CRUDs completos.
